   - 方法签名：添加`deltas`参数

## 📝 关键修改摘要

### 方法签名变化
```python
# 旧（V2）
def iter_with_primal_block(self, sample_id, alphas, betas, gammas):
def iter_with_dual_block(self, sample_id, alphas, betas, gammas):
def loss_function_differentiable(self, sample_id, alphas_tensor, betas_tensor, gammas_tensor, device):

# 新（V3）
def iter_with_primal_block(self, sample_id, alphas, betas, gammas, deltas):
def iter_with_dual_block(self, sample_id, alphas, betas, gammas, deltas):
def loss_function_differentiable(self, sample_id, alphas_tensor, betas_tensor, gammas_tensor, deltas_tensor, device):
```

### 约束表达式变化
```python
# 旧（V2）
coupling_lhs = alphas[t] * x[t] + betas[t] * x[t+1]
coupling_viol = max(0, coupling_lhs - gammas[t])

# 新（V3）
if t + 2 < self.T:
    coupling_lhs = alphas[t] * x[t] + betas[t] * x[t+1] + gammas[t] * x[t+2]
    coupling_viol = max(0, coupling_lhs - deltas[t])
```
